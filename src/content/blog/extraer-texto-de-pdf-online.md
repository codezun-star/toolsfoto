Tienes un PDF con texto y necesitas copiarlo a un Word, a una hoja de cálculo o a cualquier otro sitio. El problema es que no todos los PDFs son iguales — y el método para **extraer texto de un PDF online** depende directamente de cómo fue creado el archivo.

La herramienta [Extraer texto de PDF](/extraer-texto-pdf) de ToolsFoto funciona con PDF.js directamente en el navegador. Ningún archivo se envía a servidores externos.

## La diferencia clave: PDF nativo vs. PDF escaneado

Un **PDF nativo** fue generado directamente desde un procesador de texto, un navegador o un programa de diseño. Contiene capas de texto reales que se pueden seleccionar y buscar. Extraer el texto de estos archivos es automático y el resultado es perfecto.

Un **PDF escaneado** es básicamente una imagen dentro de un PDF. No tiene capa de texto real — cada página es una fotografía del documento. Para extraer texto de estos necesitas OCR (reconocimiento óptico de caracteres), que es una tecnología diferente.

**Cómo saber qué tipo tienes:** abre el PDF en cualquier visor e intenta seleccionar una palabra con el ratón. Si puedes seleccionarla y copiarla, es un PDF nativo. Si no puedes marcar nada, es escaneado.

## Cómo extraer el texto de un PDF paso a paso

1. Abre [Extraer texto de PDF](/extraer-texto-pdf).
2. Arrastra o selecciona tu PDF.
3. La herramienta extrae el texto de todas las páginas automáticamente.
4. Copia el texto desde la interfaz o descárgalo como archivo `.txt`.

El resultado mantiene el orden del texto tal como aparece en el PDF, incluyendo los saltos de párrafo y la separación entre secciones.

## Qué pasa si el PDF está protegido con contraseña

Si el documento tiene contraseña de apertura, la herramienta no puede procesarlo sin esa contraseña. Si solo tiene restricciones de copia (pero se abre sin contraseña), la herramienta generalmente sí puede extraer el texto igualmente.

Para PDFs con contraseña que conoces y necesitas desbloquear antes de trabajar con ellos, usa primero [Eliminar contraseña PDF](/eliminar-password-pdf) y después extrae el texto.
